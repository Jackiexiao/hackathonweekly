# 周周黑客松

周周黑客松是一个专注于AI创新的极客社区，我们致力于：
- 🚀 每周组织线下黑客松，让创新成为习惯
- 🌟 连接创新者，激发创造力
- 🔧 提供实践平台，让想法变为现实
- 🤝 建立创新生态，助力技术发展

## 我们的特色

### 🎯 定期活动
- **周周黑客松**：每周末的创新实践
- **AI共学**：定期技术分享与研讨
- **Demo Day**：项目展示与交流平台
- **AI酒馆**：轻松的创新者社交

### 👥 会员体系
- **免费会员**：参与基础活动，体验创新氛围
- **进阶会员**：获得导师指导，深度参与项目
- **导师团队**：行业专家引领，助力成长

### 🌈 发展路径
- **技术提升**：实践驱动的学习模式
- **项目孵化**：从想法到产品的全程支持
- **资源对接**：连接投资人、企业与创业者
- **社区建设**：共建创新生态圈
