# 社区愿景与使命

## 愿景（Vision）
打造一个充满创造力和活力的开源创客社区，通过技术创新推动社会进步，最终发展成为具有社会影响力的科技公益基金会。

> [基金会模式说明](foundation_model.md)

## 使命（Mission）
帮助每个参与者通过周周黑客松活动实现最小可行产品（MVP），在实践中成长，在协作中创新。我们致力于：
- 为技术爱好者提供实践和成长的平台
- 培育创新项目从想法到落地的沃土
- 建立系统化的项目孵化机制
- 推广开源创客文化，促进技术交流

## 核心价值观
- 开放共享：拥抱开源精神，鼓励知识分享
- 实践创新：强调动手实践，支持创新探索
- 互助成长：营造互帮互助的学习氛围
- 包容多元：欢迎不同背景的参与者，尊重多元观点

## 发展路径
> 详见 [社区发展计划](development_phases.md)
1. **近期**：通过周周黑客松活动，帮助参与者在一周时间内，创造一个最小可行产品，解决实际问题
2. **中期**：建立系统化的项目孵化机制，扩大社区影响力
3. **远期**：发展成为兼具公益性与创新力的科技公益基金会

## 社区性质
- 开源社区：秉承开放、透明的运营理念
- 非营利组织：以社会价值为导向
- 创新平台：提供创意碰撞和技术交流的空间
- 学习社群：强调实践学习和经验分享

## 社区定位
### 当前阶段
- **面向对象**：热爱技术、渴望实践的开发者和创新者
- **活动特色**：以周周黑客松为主要形式，强调短期交付和快速反馈
- **组织形式**：保持轻量级、灵活的社区运营模式
- **发展重点**：
  - 完善活动体系
  - 积累项目经验
  - 培养核心团队
  - 建立基础激励机制

### 未来发展
- **组织升级**：随着社区规模增长，逐步向专业化组织过渡
- **商业模式**：建立可持续的运营机制，平衡公益性和商业价值
- **社会影响**：扩大社区影响力，推动技术创新服务社会
- **生态建设**：构建开放、多元的创新生态系统

## 社区给成员的价值
- **个人成长**
  - 实践最新技术的机会
  - 项目经验积累
  - 技术能力提升

- **社交网络**
  - 认识志同道合的伙伴
  - 寻找潜在的合作者
  - 建立专业人脉

- **创新机会**
  - 参与创新项目
  - 获得项目孵化支持
  - 分享项目收益

- **社会价值**
  - 参与开源贡献
  - 推动技术创新
  - 创造社会影响

## 发展路径
社区将遵循"社区初创期 -> 成长期 -> 成熟期"的发展路径，每个阶段都有明确的目标和重点工作，确保社区的健康、可持续发展。具体发展规划参见：[发展规划](development_phases.md)

品牌故事
周周黑客松：从无聊到创造，从聊天到行动
一开始，我只是一个普通的开发者，热爱创造有趣的小产品，但也常常感到孤独和迷茫。参加了许多所谓的交流会，但大多停留在聊天和打鸡血上，真正落地的东西却少之又少。
有一天，我在想：为什么不试着做点不一样的事情？
不只是聊天，不只是分享，而是真正行动起来。组队做个东西，哪怕只是个小工具、一个初步的想法，但至少是实实在在的。于是，我组织了一场小型黑客松，邀请了几个志同道合的朋友，简单分组，用一天时间从想法到原型，把一个个念头变成现实。
出乎意料的是，这种活动不仅激发了我们的创造力，还让我们体验到了合作和成就感。更重要的是，我们创造的东西哪怕还很简单，却是独一无二的、属于我们的成果。
活动结束后，越来越多的人找我，希望能一起参与下一次活动。他们有的说：
- “这是我第一次真正把自己的点子变成了东西。”
- “我一直以为自己只能参与讨论，没想到真的能做出来。”
在这个 AI 浪潮（甚至失业潮）的时代，很多人正在寻找方向和机会。周周黑客松，成为了一个帮助人们用实践拥抱未来的平台。
今天的周周黑客松，已经不仅仅是一个人的想法，它变成了一个连接无数创造者的社区。我们聚焦于实践和落地，每个月都抽出时间，沉浸在有趣的团队合作中。这里没有对话空转的“交流会”，只有带着成果离场的满足感。
我们的信念很简单：如果你想做，就有可能实现；如果你尝试，就会比原地不动更靠近梦想。
