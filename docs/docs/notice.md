# 文档状态说明

> ⚠️ 重要提醒：本文档库中的部分内容仍处于规划和讨论阶段，请注意查看以下标记。

## 文档状态标记说明

### 🔮 远期规划
> 🔮 远期规划：此文档描述的是社区发展到一定规模后才可能实施的计划

带有此标记的文档描述了社区在发展到一定规模后可能会实施的计划和设想。这些内容：
- 需要社区发展到一定规模才能实现
- 目前仅作为未来发展方向的参考
- 具体实施方案和时间还需要根据社区实际发展情况决定

### 📝 草案讨论中
> 📝 草案讨论中：此文档描述的是近期计划，但目前仍在讨论阶段，内容可能会有较大调整

带有此标记的文档属于近期计划实施的内容，但目前处于：
- 初步构思阶段
- 正在收集社区反馈
- 细节仍在讨论和完善中
- 最终方案可能会有较大调整

## 使用方法

在相关文档的开头，我们会添加对应的标记。例如：

```markdown
# 某项目计划
🔮 远期规划

正文内容...
```

或

```markdown
# 某项目方案
📝 草案讨论中

正文内容...
```

请读者在阅读和参考这些文档时，注意查看文档状态标记，并结合实际情况理解和使用这些内容。
