# AI 酒吧

## reference
### Datawhale和年轻人的学术酒吧
> https://mp.weixin.qq.com/s/TKm9_JltsVy79phjn5DcUg

zzm 在Datawhale进行交流的过程中，发现了大家都非常渴望与不同学校、不同职业的外界的人进行链接，但是没有一个轻松的机会能让大家放心大胆的社交。因此 zzm 和大黄专门策划举办了第一场 Datawhale 学术酒吧，希望大家可以在一个放松的场合与不同的人进行连接。　

12 月 8 日下午一点，Datawhale 学术酒吧在杭州 IT 公园五楼举办，大家先进行自我介绍，后围绕职业选择与成长展开热烈讨论，人们拿着酒杯来来往往，完全没有传统讲座那般正襟危坐。　
ChatMind AI 和杭电计科协支持了调酒用具和酒，Datawhale支持了30瓶可乐，大家带着自己喜欢吃的零食来分享，并且请了咖啡师紫外线和调酒师小游来给大家调酒。　

来参与朋友非常多元，有在校的大学生（其中甚至有警察学院的），在职的工作人员，创业的大佬，大家在这里没有身份上的差异。　

在酒精的作用下，讲座那种鸦雀无声的气氛也消失了，也没有不知所措的尴尬，取而代之的是现场讨论不断，大家以一种非常轻松的姿态与不同的人进行社交，现场采用抽签的方式发起话题，大伙专门围绕“自己的如何将自己的职业和热爱相结合”，“产品经理的职能类别”，“如何看待 AI 给行业的赋能”进行了深入的讨论，大佬在现场给了很多行业内的视角，大家都受益匪浅。　

在场还有一位同学带来了自研的无人机，现场展示并且低空飞行，也找到了志同道合想要一起做无人机的朋友。　

后续 Datawhale 也考虑在其他城市举办学术酒吧，欢迎留言你所在的城市
