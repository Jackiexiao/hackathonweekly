# 周周黑客松社区项目合作协议

**版本** = "1.0.0"

**协议方**

甲方（社区方）：周周黑客松社区
以下简称"社区方"

乙方（项目负责人）：
以下简称"负责人"

**前言**

1. 社区方拥有"周周黑客松"品牌及其相关资源。
2. 负责人具备相关项目运营能力，愿意基于社区品牌开展特定项目。
3. 双方本着互利共赢的原则，就项目合作达成如下协议。

**第一条：项目说明**

1. 项目名称：_____________
2. 项目类型：□ AI酒馆 □ 社区周边 □ 社区小程序 □ AI编程课程 □ 其他________
3. 项目简介：_____________
4. 预期目标：_____________

**第二条：双方权责**

社区方责任：
1. 授权负责人使用"周周黑客松"品牌及相关标识
2. 提供社区资源支持：
   - 社区流量与用户资源
   - 社区营销渠道
   - 社区技术支持
   - 社区运营经验分享
3. 提供必要的项目指导和建议
4. 协助项目推广和运营

负责人责任：
1. 负责项目的具体实施和运营
2. 确保项目质量和服务水平
3. 维护社区品牌形象
4. 定期向社区方汇报项目进展
5. 承担项目运营成本
6. 遵守社区规范和价值观

**第三条：收益分配**

1. 基础分配比例：
   - 负责人获得项目净利润的70%
   - 社区方获得项目净利润的30%

2. 净利润定义：
   总收入扣除以下费用后的余额：
   - 直接运营成本
   - 市场推广费用
   - 技术服务费用
   - 人力成本
   - 其他必要支出

3. 特殊激励：
   当月度净利润达到以下目标时，负责人可获得额外奖励：
   - 超过10,000元：基础分配比例调整为75:25
   - 超过30,000元：基础分配比例调整为80:20

4. 收益结算：
   - 结算周期：每月结算一次
   - 结算日期：次月10日前
   - 结算方式：转账支付

**第四条：知识产权**

1. 项目期间产生的与"周周黑客松"品牌相关的知识产权归社区方所有
2. 负责人在项目中独立开发的技术、工具等知识产权归负责人所有
3. 双方共同开发的内容，知识产权归双方共有

**第五条：保密义务**

1. 双方对合作过程中知悉的对方商业秘密负有保密义务
2. 保密义务期限：合作期间及合作终止后两年内

**第六条：协议期限**

1. 本协议有效期：____年，自____年__月__日起至____年__月__日止
2. 协议到期前一个月，双方可协商续约事宜

**第七条：终止与解除**

1. 有下列情况之一的，可终止协议：
   - 双方协商一致
   - 项目连续3个月无实质性进展
   - 项目连续3个月无收益
   - 任何一方严重违反本协议约定

2. 终止流程：
   - 提前30天书面通知
   - 完成所有未结算收益的分配
   - 处理好用户权益过渡

**第八条：其他**

1. 本协议未尽事宜，双方可另行协商并签订补充协议
2. 本协议一式两份，双方各执一份，具有同等法律效力
