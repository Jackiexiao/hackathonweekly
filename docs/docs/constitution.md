# 周周黑客松社区章程

## 1. [总则](general_rules.md)

周周黑客松社区（简称“社区”），社会企业名为周周黑客松（以下简称“公司”）。

初期采用社会企业模式运营，未来目标是成立非营利性基金会作为母组织，管理并支持所有子项目与公司。


### 1.1 社区愿景
我们致力于成为中国领先的极客与创业者聚集地，通过孵化创新项目、连接全国各地俱乐部以及提供优质资源，助力技术与商业的深度融合。

### 1.2 使命
- 连接创客资源，孵化创新项目，为创客和创业者提供成长的平台
- 通过黑客松形式，推动创新实践与创客文化在全国乃至全球范围内的传播

### 1.3 服务范围
- 举办定期黑客松活动，促进创新合作与技术分享
- 支持创业项目孵化，提供初始资金与资源对接
- 建立全国性俱乐部网络，开展地方性活动
- 提供社区周边、课程等服务以支持运营

## 2. [组织架构](organization.md)

### 2.1 核心机构
- 运营委员会：负责社区战略规划与日常管理
- 决策委员会：由创始团队与核心股东组成，审议重大决策
- 城市俱乐部：各地分部由独立团队负责管理
- 顾问团队：提供战略、技术、投资等方面支持

### 2.2 核心岗位
详细的岗位说明与职责请参见：[岗位职责说明](roles_and_responsibilities.md)
- CEO（总负责人）：负责社区整体愿景、规划与资源整合
- COO（运营负责人）：负责日常运营与多城市俱乐部管理
- CTO（技术负责人）：技术平台搭建与项目技术支持
- CMO（市场负责人）：负责对外合作与市场推广
- 财务负责人：预算管理、财务审计与分账监督
- 地方负责人：本地俱乐部活动组织与财务报备

## 3. [俱乐部制度](club_system.md)

### 3.1 独立性与管理
- 城市俱乐部具备一定独立性，可单独组织活动并管理财务
- 收入分成：俱乐部按收入比例（10%-20%）上交总部

### 3.2 运营支持
- 总部提供统一的品牌、技术和运营支持
- 非公司化俱乐部由总部代管财务，逐步过渡至独立管理

## 7. [股权与激励机制](equity.md)
### 7.1 股权分配

- 设置预留股份（20%-30%）用于后期激励与非营利组织持股。
- 创始团队根据投入资金与贡献比例分配初始股权。
- 全职核心成员按固定比例增长股权。
- 重大贡献者通过绩效评估获取额外股权奖励。
- 定期评估团队成员贡献，使用股权池动态调整。

### 7.2 利润分配
- 孵化项目盈利后分配20%-30%给社区总部
- 利润优先用于支持非营利活动与新项目孵化

## 13. [退出机制](exit_mechanism.md)

### 13.1 创业失败退出
- 社区关闭需清算所有财务账目，公平分配剩余资产
- 核心股东与团队优先偿还初始投资
- 清算后剩余资产优先用于非营利基金会建设

### 13.2 团队成员退出
- 退出成员可按股权价值回购股份
- 核心岗位退出需提前3个月申请并完成交接
- 创始团队退出需经过运营委员会审批，转让股份需遵循相关协议

### 13.3 社区解散
- 解散需由决策委员会与核心股东联合批准
- 需完成所有项目的清算和交接工作
- 剩余资产优先用于非营利基金会建设

## FAQ

### 合适的融资时间
融资时机取决于项目的发展阶段和资金需求，建议分为以下几个阶段：

探索阶段（当前阶段）

以创始人投入为主，同时探索盈利项目的可行性。
建议不引入外部投资者，而是专注于验证模式和构建社区基础。
验证阶段（盈利模式初步明确，社区影响力扩大）

可考虑通过引入天使投资或小规模战略投资，解决运营资金问题。
投资金额应与实际需求匹配，避免过度融资。
扩展阶段（盈利模式成熟，计划快速扩张）

适合进行更大规模的融资，比如Pre-A轮或A轮融资，用于拓展市场、升级产品等。
此阶段融资的目标是扩大规模，而不是解决短期资金问题。


投资与赞助的整合建议

1. 前期资金来源的组合
创始人自筹 + 小规模赞助（冠名、设备支持）。
如果资金仍有缺口，可以通过以下方式解决：
与目标明确的天使投资者合作（如对社区型项目感兴趣的投资者）。
小额众筹（结合周边产品销售等方式）。

2. 股权预留与管理
预留5%-10%股权给未来可能成立的非营利性组织，用于公益项目管理。
设立赞助专项基金：接受企业或个人赞助，但明确规定资金仅用于公益活动。

3. 盈利项目的作用
盈利性项目（如 AI 酒馆、社区周边）不仅是资金来源，也可用于测试市场和提高社区影响力。

早期盈利项目的收益建议按以下分配：
**60%-70%**用于项目再投资和社区发展。
**30%-40%**分配给核心创始团队或负责人，用于激励。

4. 未来融资规划
当盈利项目开始产生稳定现金流时，再进行外部融资，以合理的估值吸引战略投资者。
保留创始团队对社区的控制权（>51%股权），确保长期发展方向不偏离初衷。
