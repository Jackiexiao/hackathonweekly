 日常活动详细SOP
- 日常活动（AI共学）
  - 前期准备
    - T-7天
      - 确定主题和主讲人
      - 发布活动预告
      - 准备宣传文案
    - T-3天
      - 参与者提醒
      - 收集预习资料
      - 准备实践案例
    - T-1天
      - 场地设备确认
      - 物料准备就绪
  - 现场执行
    - 13:30-14:00 签到入场
    - 14:00-14:15 开场介绍
    - 14:15-15:30 主题分享
    - 15:30-15:45 茶歇交流
    - 15:45-16:30 实践环节
    - 16:30-17:00 项目展示
    - 17:00-17:30 总结反馈

- Demo Day
  - 前期准备
    - T-14天
      - 收集项目报名
      - 确定展示名单
    - T-7天
      - 项目预演
      - 邀请嘉宾
    - T-3天
      - 场地布置方案
      - 设备测试
  - 现场执行
    - 13:30-14:00 签到布展
    - 14:00-14:30 开场致辞
    - 14:30-16:00 项目展示
    - 16:00-16:30 茶歇交流
    - 16:30-17:30 专家点评
    - 17:30-18:00 颁奖总结

- AI馆
  - 前期准备
    - T-7天
      - 确定主题与主理人
      - 邀请特邀嘉宾
    - T-3天
      - 场地预订确认
      - 餐饮安排
  - 现场执行
    - 19:00-19:30 签到入场
    - 19:30-20:00 主题分享
    - 20:00-21:00 自由交流
    - 21:00-21:30 项目推介
    - 21:30-22:00 总结离场
