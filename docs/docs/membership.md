# 会员制度

## 会员等级体系

### 免费会员
1. **探索者 (Lv.0)**
   - **加入条件**：完成注册，认同社区价值观
   - **核心权益**：
     - 参与基础社区活动
     - 浏览项目展示区
     - 基础社区互动权限
     - 获得"探索者"徽章

2. **创客新星 (Lv.1)**
   - **加入条件**：
     - 参与2次线下活动
     - 在社区发表首个项目想法
     - 累计50积分
   - **核心权益**：
     - 探索者权益+
     - 获得"创客新星"徽章
     - 可进行项目Demo展示
     - 获得社区基础反馈

3. **实践者 (Lv.2)**
   - **加入条件**：
     - 完成首个MVP演示
     - 参与1次志愿服务
     - 累计150积分
   - **核心权益**：
     - 创客新星权益+
     - 获得"实践者"徽章
     - 加入志愿者团队资格
     - 项目获得重点推荐
     - 可申请升级付费会员

### 付费会员（299元/半年）
1. **创新达人 (Lv.3)**
   - **加入条件**：
     - 完成半年会费缴纳
     - 2次Demo展示经历
     - 累计300积分
   - **核心权益**：
     - 实践者权益+
     - 获得"创新达人"徽章
     - 月度导师指导机会
     - 优先参与特色活动
     - 专属项目展示机会

2. **创业先锋 (Lv.4)**
   - **加入条件**：
     - 持续会员满1年
     - 累计500积分
     - 项目达到以下任一标准：
       - 月活用户>500
       - 月营收>5万
     - 指导2个社区项目
   - **核心权益**：
     - 创新达人权益+
     - 获得"创业先锋"徽章
     - 周度导师指导
     - 项目孵化支持
     - 商业资源对接
     - 参与社区决策建议

3. **社区导师 (Lv.5)**
   - **加入条件**：
     - 累计1000积分
     - 项目达到以下任一标准：
       - 年营收>50万
       - 用户量>5000
     - 培养2个成功项目
   - **核心权益**：
     - 创业先锋权益+
     - 获得"导师"徽章
     - 社区重大决策权
     - 导师收益分成
     - 项目孵化基金支持
     - 社区品牌背书

## 快速晋升通道

### 资深从业者直通Lv.3（需缴费）
满足以下任一条件：
- 3年以上相关行业经验
- 主导过成功商业项目
- 知名公司核心工程师
- 有影响力的技术博主

### 创业者直通Lv.4（需缴费）
满足以下任一条件：
- 月收入>1万的项目
- 获得过天使投资
- 产品用户量>1000
- 成功exits经历

### 导师特批直通Lv.5（免会费）
满足以下任一条件：
- 行业专家或知名创业者
- 知名加速器/VC合伙人
- 大厂专家
- 成功创业者（估值>500万）

## 积分体系

### 基础活动积分
- 参与线下活动：10分/次
- 迷你黑客松展示：15分/次
- 志愿服务：20分/次
- 成功组队开发：20分/次

### 项目相关积分
- 完成MVP：30分
- 项目被收录：25分
- 项目月活破百：40分
- 获得投资：100分

## 社区徽章体系

### 成就徽章
1. **创造者系列**
   - 初创者 (完成首个MVP)
   - 创新者 (项目获得用户反馈)
   - 连续创造者 (3个以上项目)
   - 创新先锋 (项目营收破万)

2. **贡献者系列**
   - 志愿先锋 (50小时志愿服务)
   - 知识传道者 (10次技术分享)
   - 社区建设者 (年度贡献top10)
   - 优秀导师 (指导3个成功项目)

3. **里程碑系列**
   - 活动达人 (参与20次活动)
   - 项目达人 (完成3个项目)
   - 引路者 (帮助3个新人)
   - 资源达人 (引入5个有效资源)

### 特殊徽章
- 年度之星
- 最佳项目
- 社区贡献奖
- 创新突破奖

## 导师管理

### 导师类型
- 技术导师
- 创业导师
- 行业专家
- 社区深度会员

### 导师权益
- 课程收入分成（70%）
- 项目孵化收益分成（30%）
- 社区资源优先使用权
- 品牌背书与推广

### 导师义务
- 每月至少一次分享或指导
- 参与社区重要决策
- 帮助审核项目质量
- 以低于市场价的价格或公益形式提供指导服务
