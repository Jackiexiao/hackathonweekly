**前言**

1. 创始人希望创建产品/服务并将其推向市场。
2. 孵化方愿意以收益分成贷款的形式为创始人提供特定支持和资金，这些支持/资金将根据本协议由孵化方投资。
3. 孵化方以收益分成贷款的形式向创始人提供支持和资金。作为回报，创始人同意与孵化方分享项目产生的一定比例收益。

**条款**

**支持内容：**

孵化方向创始人提供价值**30,000元**的支持，具体包括：

1. 培训资源（价值5,000元）：
   1. 访问全面的知识库和行业洞察（1,000元）
   2. 参与密集型训练营项目（2,500元）
   3. 访问历史孵化项目的案例和数据（1,000元）
   4. 行业专家的个性化指导（500元）

2. 渠道和人脉资源（价值10,000元）：
   1. 主要平台上的战略性产品发布支持（7,000元）
   2. 获取独家产品发布网站和平台（1,000元）
   3. 对接孵化方的投资者、合作伙伴和潜在客户网络（1,500元）
   4. 参与社交活动和创始人见面会（500元）

3. 增值服务（价值12,000元）：
   1. 全面的创意验证和市场研究（2,000元）
   2. 产品落地页的专业修改和优化（1,500元）
   3. 定向付费广告支持和策略（2,000元）
   4. 支付系统搭建和优化协助（1,500元）
   5. 定制营销策略制定和执行支持（2,000元）
   6. 深度SEO分析和优化（1,500元）
   7. 用户体验（UX）评估和建议（750元）
   8. 法律和合规指导（750元）

4. 技术和全球市场支持（价值3,000元）：
   1. 获取前沿AI工具和资源（1,000元）
   2. 技术架构评估和优化（750元）
   3. 文化适应和本地化指导（625元）
   4. 全球品牌建设支持（625元）

创始人可以根据本协议条款申请额外的资金或支持。孵化方保留批准或拒绝任何此类申请的权利。

**收益分成：**

作为孵化方提供支持的对价，创始人同意与孵化方分享项目产生的收益。收益分成比例如下：

- 月度经常性收入（MRR）在50,000元（含）以下部分的20%
- 月度经常性收入（MRR）超过50,000元部分的10%

举例说明，如果月度经常性收入为80,000元，收益分成计算如下：
(20% × 50,000元) + (10% × 30,000元) = 10,000元 + 3,000元 = 13,000元

**收益上限：**

孵化方的收益分成设有**5倍**上限。这意味着创始人向孵化方分享的总收益不会超过初始支持/投资的5倍。

当达到5倍上限时，创始人与孵化方之间的收益分成协议即告完成。此时，创始人不再有义务与孵化方分享进一步的收益，孵化方也不得根据本协议提出任何额外的索赔或权利。

如果项目在本协议开始后18个月内被出售，收益分成上限将降低至3倍。

如果项目在收益分成模式完全完成之前被出售，孵化方将获得售价的**20%**。如果售价计算出的分成超过达到调整后上限所需的剩余金额，孵化方的分成将限制在完成调整后上限所需的金额。

举例说明：
- 如果产品在1年后以50,000元的价格出售，孵化方将获得10,000元（50,000元的20%）
- 如果产品在3年后以1,000,000元的价格出售，孵化方将获得150,000元（30,000元的5倍，如有后续投资，上限将相应调整）

**孵化营：**

创始人应积极参与孵化方提供的孵化营。参与条款如下：

- 加入孵化营时，创始人需支付**999元**押金。此押金旨在确保创始人在孵化期间的投入和努力。
- 如果创始人在孵化期结束时达到预定目标，押金可退还。具体目标将由创始人和孵化方共同确定。

**抵押要求：**

双方确认并同意本收益分成协议无需抵押。

**监督：**

创始人将投入所有必要资源并采取所有必要行动来维护自身及孵化方的利益。

**信息义务：**

创始人应向孵化方提供关于产品状态的充分信息。

**签署：**

兹证明，协议各方已于上述日期签署本收益分成协议。
