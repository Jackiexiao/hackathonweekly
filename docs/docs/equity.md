# 股权与决策机制设计方案

## 前言
在构建初始团队和确定股权分配、决策机制时，需要兼顾以下几方面：
- 贡献类型
- 长期激励
- 公平性
- 灵活性

这些因素对确保团队的稳定性和社区的发展方向至关重要。

## 1. 初始团队股权分配

### 1.1 股权分配原则

#### 贡献类型分层
- **资金投入者（投资型）**：根据实际资金投入分配相应股权
- **核心贡献者（劳动型）**：根据工作内容和贡献价值给予股权，适用于全职或重要兼职的组织者
- **混合型贡献者**：既投入资金又有重要劳动贡献的人，需结合资金比例和劳动价值综合分配股权

#### 股权比例建议
- 资金投入者：50%-60%（资金占大头，但不应让投资者完全控制）
- 核心贡献者：30%-40%（为实际出力的人提供足够激励）
- 预留股份：10%-20%（用于未来吸引新成员或为社区公益性组织预留）

#### 时间与流动性要求
- **股权归属计划**：4年归属期+1年锁定期，确保成员长期投入
- **退出机制**：核心组织者退出或贡献减少时，可通过回购机制收回部分股权

## 2. 核心股东标准与职责

### 2.1 核心股东标准
- **资金投入门槛**：最低投资额度10万人民币以上
- **贡献门槛**：至少负责1年以上的重大运营职责
- **混合贡献**：投入资金和劳动达到一定比例者优先考虑

### 2.2 核心股东职责
1. **战略决策**：参与公司/社区的方向性讨论
2. **资源支持**：提供资金、渠道或市场支持
3. **执行监督**：监督各地俱乐部及项目执行进度
4. **长期投入**：签订协议，明确至少3年服务期限

## 3. 决策委员会的组成与机制

### 3.1 决策委员会组成
- **人数**：5-7人
- **成员来源**：
  - 核心创始团队成员
  - 城市负责人代表
  - 特定专业领域代表

### 3.2 决策机制
1. **重大事项投票决策**
   - 需委员会决策的事项清单
   - 采用简单多数（>50%）或超级多数（>66%）规则
2. **日常运营授权**
   - 日常事务授权给具体负责人执行
   - 委员会定期监督
3. **创始人一票否决权**
   - 前期阶段保留创始人一票否决权

## 4. 分利润与薪资机制

### 4.1 利润分成机制
- **总部与地方收入分成**
  - 地方俱乐部上交10%-20%收入至总部
  - 地方俱乐部独立管理剩余收入
- **核心股东分红**
  - 年度净利润30%-40%用于分红
  - 剩余利润用于再投资

### 4.2 工资发放机制
- **全职组织者**
  - 市场水平工资（月薪5000-15000元）
  - 工资与分红独立发放
- **兼职组织者**
  - 按小时/项目制发放酬劳
  - 长期兼职者可获得利润分成或绩效奖金

## 5. 前期未成立公司的俱乐部处理

### 5.1 临时管理模式
- 以总部名义运营
- 财务统一管理
- 设定收入上报比例

### 5.2 未来成立计划
- 预留部分股份（约10%）支持优秀俱乐部
- 协助完成注册并实现财务独立
- 保留总部收入分成机制
