# 俱乐部制度

## 俱乐部组织架构

### 主理人制度

#### 常驻主理人
- **职位设置**
  - 俱乐部主席：负责俱乐部整体运营和发展
  - 活动主管：负责活动策划和执行
  - 技术主管：负责技术指导和项目评审
  - 运营主管：负责会员服务和日常运营

- **职责要求**
  1. 俱乐部主席
     - 制定俱乐部发展规划
     - 协调各部门工作
     - 对接总部资源
     - KPI：会员增长率、活动质量评分

  2. 活动主管
     - 活动策划与执行
     - 场地资源协调
     - 活动数据统计
     - KPI：活动频率、参与度、满意度

  3. 技术主管
     - 技术分享组织
     - 项目评审与指导
     - 技术资源对接
     - KPI：技术分享场次、项目完成率

  4. 运营主管
     - 会员服务与管理
     - 社群运营维护
     - 内容运营与宣传
     - KPI：会员活跃度、内容产出量

#### 流动主理人
- **晋升路径**
  1. 志愿者阶段
     - 参与活动协助
     - 累计志愿服务时长
     - 收集会员反馈

  2. 见习主理人
     - 协助常驻主理人工作
     - 独立组织小型活动
     - 参与俱乐部决策

  3. 正式主理人
     - 可独立发起活动
     - 获得活动预算
     - 参与重要决策

- **晋升要求**
  - 志愿者→见习主理人
    - 完成3个月志愿服务
    - 获得2位常驻主理人推荐
    - 通过总部面试

  - 见习主理人→正式主理人
    - 独立组织3次成功活动
    - 获得会员好评
    - 通过总部认证

## 俱乐部会员体系

### 会员等级
1. **普通会员**
   - 参与基础活动
   - 使用公共资源
   - 社群互动权限

2. **进阶会员**
   - 优先参与活动
   - 专属资源使用
   - 项目展示机会
   - 导师指导机会

3. **核心会员**
   - 参与俱乐部决策
   - 获得项目孵化支持
   - 优先使用所有资源
   - 成为导师候选人

  - 活动参与积分

  - 志愿服务积分
  - 社群互动积分

- **等级晋升**
  - 基于积分达标
  - 需完成指定任务
  - 通过评审认证
